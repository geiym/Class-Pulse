# ClassPulse

ClassPulse is a lightweight, standalone Android application designed to help students track and reflect on their academic engagement, subject schedules, and class participation habits.

## Project Specifications
* **Architecture:** Offline-first standalone application (Room Database / MVVM Architecture)
* **Development IDE:** Android Studio
* **Target OS:** Android 8.0 (Oreo) or higher

---

## Key Features & Layouts

* **1. Class Setup & Schedule Management**
  * Effortlessly manage current subjects, class timings, and instructor profiles.

* **2. Post-Class Reflection Log**
  * Log attendance status, track distinct participation levels (*Low, Medium, or High*), and save context notes on topics discussed.

* **3. Engagement Dashboard & Trends**
  * View intuitive visualizations of weekly/monthly academic reflection data with rule-based feedback insights pointing out focus habits.
